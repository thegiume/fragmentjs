API Reference

Full control in a few functions.

FragmentJS APIs are intentionally minimal. Here you will find the public methods and config options.

Core methods

FragmentJS.navigateTo(path, delay)

Updates the URL and shows the route sections.

path: string delay: number

FragmentJS.refresh(delay)

Reloads the current route without changing the URL.

delay?: number

FragmentJS.loadIncludes()

Re-reads all elements with the fragment attribute not yet included.

auto-start idempotente

FragmentJS.parseRoutePath(rawPath)

Normalizes a URL into pathname, query, and hash.

return: object

Includes and utilities

FragmentJS.fetchInclude(file, element, callback)

Loads an HTML file and replaces the original element.

FragmentJS.fetchHtml(url, callback)

jQuery shortcut to load HTML from an endpoint.

FragmentJS.fetchJson(url, callback)

Requests JSON and passes the data to the callback.

FragmentJS.replaceAllLiteral(input, search, replacement)

Utility for literal string replacements.
